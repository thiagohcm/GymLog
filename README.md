# GymLog - Gerenciador de Exercícios

GymLog é um aplicativo Android para gerenciar exercícios de academia, permitindo que usuários cadastrem, editem e organizem seus exercícios favoritos.

## Funcionalidades

1. **Lista de Exercícios**: Exibe todos os exercícios cadastrados com seus detalhes e permite busca por nome, categoria ou observações.

2. **Cadastro de Exercícios**: Permite adicionar novos exercícios com nome, categoria, séries, repetições e observações.

3. **Edição de Exercícios**: Possibilita modificar as informações de exercícios já cadastrados mantendo o estado de favorito.

4. **Sistema de Favoritos**: Permite marcar/desmarcar exercícios como favoritos, que aparecem prioritariamente na listagem.

5. **Exclusão de Exercícios**: Permite remover exercícios com confirmação do usuário.

## Tecnologias Utilizadas

- **Kotlin**: Linguagem principal de desenvolvimento
- **AndroidX**: Componentes modernos do Android
- **Room**: Persistência de dados local
- **ViewModel**: Gerenciamento de estados da UI
- **ViewBinding**: Vinculação segura de views
- **Coroutines**: Programação assíncrona
- **Navigation Component**: Navegação entre telas
- **Dagger Hilt**: Injeção de dependências
- **Material Design**: Componentes visuais modernos
